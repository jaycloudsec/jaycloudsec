## What I've Built

### Azure SOC Lab Series
**[→ View Labs](https://github.com/jaycloudsec/cloud-security-labs/tree/main/Azure-SOC-Lab) | [→ Read Write-ups](https://jaycloudsec.github.io/writeup.html)**

5-lab cloud security project built in Microsoft Azure using Microsoft Sentinel.

| Lab | Focus |
|-----|-------|
| Lab 00 | SIEM environment setup — Sentinel, Log Analytics, Azure Monitor Agent |
| Lab 01 | SOC incident investigation — KQL log analysis, entity pivoting |
| Lab 02 | Detection engineering — scheduled KQL analytics rule for brute force |
| Lab 03 | Honeypot threat intelligence — 64,760+ real attacks, botnet identified |
| Lab 04 | Automated SOC response — Logic App playbook, 21/21 automated alerts |

**Key finding from Lab 03:** Deployed a publicly exposed honeypot VM and collected real attack data. All six top attacking IPs traced back to a single coordinated botnet operating from AS211736 in Amsterdam, Netherlands — peak volume of ~37,000 attempts per hour.
